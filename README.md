# Machine Learning Practice

Kumpulan latihan dan eksperimen Machine Learning menggunakan Python dan Jupyter Notebook. Repository ini disusun secara sistematis mulai dari dasar pemrosesan data hingga implementasi model yang lebih kompleks.

---

## 📂 Struktur Folder

Proyek ini dibagi menjadi beberapa bagian utama:

- **`01_pandas/`**
  - Latihan dasar manipulasi data, filtering, dan agregasi menggunakan library Pandas.
- **`02_numpy/`**
  - Dasar-dasar komputasi numerik dan operasi array menggunakan library NumPy.
- **`03_eda/`**
  - *Exploratory Data Analysis* (EDA) untuk memahami pola, anomali, dan distribusi data sebelum pemodelan.
- **`04_visualization/`**
  - Teknik visualisasi data menggunakan library Matplotlib dan Seaborn untuk presentasi data yang informatif.
- **`05_classification/`**
  - Implementasi algoritma klasifikasi (misal: Logistic Regression, KNN, Decision Tree) untuk memprediksi kategori.
- **`06_clustering/`**
  - Implementasi algoritma pembelajaran tanpa pengawasan (*unsupervised learning*) seperti K-Means.
- **`07_kaggle_projects/`**
  - Kumpulan project dari kompetisi Kaggle yang mencakup Regression, Classification, dan Clustering pada dataset dunia nyata.

## 🚀 Cara Menggunakan

### 1. Clone Repository
```bash
git clone https://github.com/leapwithluvi/machine-learning-practice.git
```

### 2. Gunakan di Google Colab (Rekomendasi)
1. Buka [Google Colab](https://colab.research.google.com/).
2. Pilih tab **GitHub** dan masukkan URL repository ini.
3. Atau pilih **File → Open Notebook → Upload** dan unggah file `.ipynb` dari komputer Anda.

### 3. Jalankan secara Lokal
1. Pastikan Anda sudah menginstal Python 3.x.
2. Instal library yang dibutuhkan:
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn
   ```
3. Masuk ke folder project dan jalankan Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
